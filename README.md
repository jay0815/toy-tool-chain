# toy-tool-chain

### What's this

A automatic tool chain generator by yeoman. It can help init a project easlier and faster

### Before use

```
npm install -g yo

or

yarn add global yo
```

## How to use

* first
```
  yarn install

  or

  npm install
```

* second
```
  yarn link

  or

  npm link
```

* third

create a folder where your want to init project

then go to your created new folder

then run
```
  yo toytool 
```
